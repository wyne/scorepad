# ScorePad

Landing page for [ScorePad with Rounds](https://www.scorepadapp.com) — a free score-keeping app for board games, card games, and any competitive activity.

## About

ScorePad lets you track points across multiple players and rounds without typing numbers. Tap to add points, hold to add larger values, and view score history per round.

Available on:
- [iOS (App Store)](https://apps.apple.com)
- [Android (Google Play)](https://play.google.com)
- Web

## Tech Stack

- Static HTML/CSS/JS — no build step required
- Hosted on GitHub Pages (main branch) at [www.scorepadapp.com](https://www.scorepadapp.com) — [Pages settings](https://github.com/wyne/scorepad/settings/pages)
- Google Analytics for usage tracking

## Development

Since this is a static site, no build process is needed. Open `index.html` directly in a browser or serve the directory locally:

```bash
npx serve .
```
